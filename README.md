# lotto-generator
A Simple Generator for random Lottery numbers for EuroJackpot and the German Lottery 6 of 49

## Usage
    Usage of lotto-generator:
      -l string
    	    Kind of Lottery: 6of49 or euro (default "6of49")
      -n int
    	    Amount of genereated lottery fileds (default 1)
      -s	Genereate SuperZahl for 6/49 Lottery
